# Promise-SNS-Project

<br>

## 주요 기능

- 게시글 기능
  - 게시글 CRUD
  - 게시글 좋아요
  - 해시태그 기능
    - 해시태그 검색
    - 해시태그 팔로우 후 모아보기

- 프로필 기능
  - 프로필 수정
  - 프로필 사진 설정

- 소셜 기능
  - 타 이용자 팔로우 등록/해제
  - 프로필 페이지
    - 팔로워 팔로잉 수 확인
    - 방명록 작성 및 수정/삭제
<br><br>

## 기술 스택

- Node.js Express 
- HTML, CSS, JS
- Nunjucks, Sequelize
